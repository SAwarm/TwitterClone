## Como instalar

Clone da aplicação para alguma pasta de sua escolha

```
$ git clone https://github.com/SAwarm/TwitterClone.git
```

Entre nas pastas

```
$ cd TwitterClone
$ cd public
```

Inicialize o servidor para usar

```
$ php -S localhost:8080
```
